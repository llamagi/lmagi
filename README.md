# LINUX INSTALL

sudo apt install git

```bash
git clone https://github.com/easyGLM/ezAGI/
cd ezAGI
python3 -m venv agi
source agi/bin/activate
pip install -r requirements.txt
# activate ezAGI.py with internal reasoning (EXPERIMENTAL)
python3 ezAGI.py
```
